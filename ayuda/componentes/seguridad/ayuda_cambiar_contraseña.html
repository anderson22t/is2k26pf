Cambio de Contraseña

Paso 1: Ingresar contraseña actual

Primero debe ingresar la contraseña actual del usuario.

Paso 2: Ingresar nueva contraseña

Luego deberá ingresar la nueva contraseña que desea establecer.

Paso 3: Confirmar contraseña

Posteriormente debe confirmar la nueva contraseña para evitar errores.

Paso 4: Mostrar contraseña

Si marca la opción Mostrar contraseña, podrá visualizar la contraseña actual y la nueva de mejor manera.

Paso 5: Cambiar contraseña

Finalmente, presione el botón Cambiar. El sistema mostrará un mensaje indicando que la contraseña se cambió correctamente.
