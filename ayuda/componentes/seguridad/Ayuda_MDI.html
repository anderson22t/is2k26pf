Pantalla Principal del Sistema

¿Qué es esta pantalla?

Esta es la pantalla principal del sistema. Desde aquí el usuario puede acceder a todas las opciones disponibles utilizando el menú superior.

Menú principal

En la parte superior de la pantalla se encuentra el menú principal. Este menú contiene todas las opciones que permiten utilizar el sistema.

Opción Catálogos

Al seleccionar la opción Catálogos, el sistema mostrará una lista de formularios para ingresar y actualizar información básica, como usuarios, perfiles u otros datos necesarios.

Al hacer clic en alguna opción, se abrirá el formulario correspondiente en la pantalla.

Opción Procesos

La opción Procesos contiene los formularios que ejecutan acciones específicas del sistema.

Al hacer clic en una opción, el sistema mostrará el formulario necesario para realizar el proceso.

Opción Herramientas

En la opción Herramientas se encuentran funciones adicionales del sistema, como el cambio de contraseña.

Seleccione la herramienta que desea utilizar para continuar.

Opción Asignaciones

Al seleccionar la opción Asignaciones, se mostrarán los formularios que permiten relacionar información, por ejemplo, asignar perfiles a usuarios.

Seleccione la opción deseada para abrir el formulario correspondiente.
